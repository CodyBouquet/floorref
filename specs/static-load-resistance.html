Indentation & Static Load Resistance Explained LVT / LVP • What the Test Measures • What It Doesn’t

Static load and indentation resistance are common performance-related specs listed on vinyl flooring technical data sheets. This page explains what these measurements actually describe, how they’re tested, and why they should be interpreted carefully. Reference-only: no product or installation advice.

Quick answer

Static load resistance measures how much a floor deforms when a specific weight is applied for a specific amount of time under controlled lab conditions. It helps describe how a product behaves under sustained pressure, but it does not guarantee dent-free performance in real use.

How Static Load / Indentation is Typically Described

Item What it refers to Why it matters Limitations
Static Load A defined weight applied to the surface for a defined duration. Helps describe sustained pressure behavior. Does not account for moving or uneven loads.
Indentation Depth Measured deformation after the load is removed. Shows how much the surface changed shape. Measured under specific lab conditions.
Test Duration How long the load is applied (hours or days). Longer durations simulate sustained pressure. Real-world loads vary widely.
Recovery Time Time allowed for the material to rebound. Shows elastic vs permanent deformation. Recovery varies by material and environment.

Quick explainers

What Static Load Tests Actually Simulate

Static load tests simulate a fixed weight pressing on a small surface area for a set amount of time. They are useful for controlled comparisons, but they do not simulate rolling loads, furniture movement, or point impacts.

Why Results Vary Between Products

Results depend on core type, density, thickness, backing layers, and formulation. Two floors with the same core label (SPC/WPC) can show different indentation results.

Static Load vs Real-World Use

Real-world conditions involve uneven loads, temperature changes, and movement. Static load values should be treated as descriptive data, not performance guarantees.

FAQ

Is a lower indentation number always better? ⌄

Generally lower deformation values indicate less visible indentation under test conditions, but they must be compared using the same test method and conditions.

Do static load ratings apply to glue-down LVT? ⌄

Static load tests are more commonly associated with rigid core/floating products, but similar indentation concepts may appear on some glue-down technical sheets.
