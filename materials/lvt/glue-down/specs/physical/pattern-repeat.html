Spec Sheet Details to Look For

Physical details affect takeoffs, handling, and visual planning.

  • Pattern repeat may be stated as a repeat length or the number of unique faces per carton.
  • Confirm the unit of measure and whether the value is listed as nominal, minimum, or tested result.
  • Look for the test standard or method cited; results are only comparable when the method matches.
  • Check whether the value applies to the full assembly (plank + underlayment + subfloor) or a single layer.

Planning Checklist

  • Check the repeat length or the number of unique faces.
  • Confirm whether the repeat is per carton or per pallet.
  • Review layout guidance to avoid visible repetition.
  • Consider plank size and room width for visual balance.
  • Match embossing (EIR) to printed patterns when applicable.

Common Pitfalls

  • Assuming a large repeat eliminates pattern matching needs.
  • Overlooking how directionality affects repetition.
  • Mixing cartons without shuffling planks.

FAQ

Is pattern repeat the same as number of faces? ⌄

They are related. The repeat may be expressed as a length or the count of unique visuals.

Does pattern repeat affect performance? ⌄

Not directly; it is primarily a visual specification.
