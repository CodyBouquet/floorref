Deep Dive

Construction snapshot

Core type describes the main structural layer in many LVT products, such as rigid SPC, WPC, or flexible vinyl constructions, which influences stiffness and stability.

This spec sits in the construction block because it influences how the plank is built and how it feels underfoot.

How it shows up on data sheets

Manufacturers often list core type next to the wear layer, total thickness, and backing details so the layer stack is easy to compare.

Core type alone does not define quality; compare core thickness, density, and stability metrics together.

What to compare alongside it

Review total thickness, locking system, and indentation resistance to understand how the core performs in the full assembly.

These companion specs confirm whether the construction supports the performance claims.

Spec Sheet Details to Look For

Construction details show how the product is built and how the layers interact.

  • Core type is usually described as SPC, WPC, or flexible LVT, sometimes without a standardized density value.
  • If density is listed, use it alongside thickness to understand stiffness and point-load performance.
  • Confirm the unit of measure and whether the value is listed as nominal, minimum, or tested result.
  • Look for the test standard or method cited; results are only comparable when the method matches.
  • Check whether the value applies to the full assembly (plank + underlayment + subfloor) or a single layer.

Construction Checklist

  • Identify whether the product is rigid core (SPC/WPC) or flexible LVT.
  • Check density or core thickness if provided.
  • Review dimensional stability and impact/indentation tests.
  • Confirm locking system compatibility for rigid core planks.
  • Note if an attached pad is included.

Common Pitfalls

  • Assuming SPC always outperforms WPC without looking at other specs.
  • Comparing core types without matching total thickness or wear layer.
  • Ignoring installation requirements tied to core stiffness.

FAQ

Is SPC always better than WPC? ⌄

Not inherently. Core type is one part of the construction; performance depends on the full system.

Does core type matter for glue-down LVT? ⌄

Glue-down LVT is typically flexible, so core type references may differ from rigid core products.
