Deep Dive

Documentation overview

Phthalate-free indicates the product does not use certain phthalate plasticizers often associated with PVC formulations, typically verified by testing or certifications.

Environmental specs are typically documentation-driven and tied to project submittals.

Why it matters for projects

Many projects specify phthalate-free materials for indoor air quality or compliance with health standards.

Compare like-for-like specs with the same test method, units, and installation system to avoid misleading conclusions.

Related documentation

Review phthalate-free claims alongside VOC certifications, HPDs, and recycled content disclosures.

Check dates, scope, and whether the document is product-specific.

Spec Sheet Details to Look For

Environmental detail sections clarify documentation scope and compliance boundaries.

  • Phthalate-free claims may list specific restricted chemicals or reference third-party testing.
  • Confirm the unit of measure and whether the value is listed as nominal, minimum, or tested result.
  • Look for the test standard or method cited; results are only comparable when the method matches.
  • Check whether the value applies to the full assembly (plank + underlayment + subfloor) or a single layer.

Documentation Checklist

  • Check which phthalates are excluded (e.g., DEHP, BBP, DBP).
  • Look for third-party testing or certification evidence.
  • Confirm if the claim applies to all layers or just the wear layer.
  • Match the claim to project or regulatory requirements.
  • Request updated documentation if versions change.

Common Pitfalls

  • Assuming phthalate-free automatically means low VOCs.
  • Using outdated documentation without checking dates.
  • Comparing claims that define “phthalate-free” differently.

FAQ

Is phthalate-free the same as PVC-free? ⌄

No. Products can still be PVC-based while avoiding phthalate plasticizers.

Do all LVT products disclose phthalate status? ⌄

Not always; you may need to request documentation or an HPD.
