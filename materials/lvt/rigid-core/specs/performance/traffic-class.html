Spec Sheet Details to Look For

Performance detail sections explain how results were generated and what they actually indicate.

  • Traffic class is commonly listed as ISO/EN classes (e.g., 23/31/33) with descriptions of intended use levels.
  • Confirm the unit of measure and whether the value is listed as nominal, minimum, or tested result.
  • Look for the test standard or method cited; results are only comparable when the method matches.
  • Check whether the value applies to the full assembly (plank + underlayment + subfloor) or a single layer.

Performance Checklist

  • Identify the rating system used (ISO 10874/EN 685).
  • Match the class to the intended space type.
  • Check if ratings differ for residential vs commercial use.
  • Confirm wear layer and indentation specs that support the class.
  • Review warranty coverage tied to the use class.

Common Pitfalls

  • Assuming a higher class means suitability for any environment.
  • Ignoring rolling load or chair caster requirements.
  • Comparing classes across different standards.

FAQ

Is traffic class the same as AC rating? ⌄

No. AC rating is primarily for laminate abrasion; traffic class is a broader use classification.

Do all LVT products have traffic class ratings? ⌄

Not always; some manufacturers use different classification systems or omit it.
