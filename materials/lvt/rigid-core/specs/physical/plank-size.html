Plank Size Explained Length · Width · Nominal vs. Actual · Visual Scale · Subfloor Impact

Plank size specifies the nominal length and width of a rigid core LVT plank or tile, typically listed in inches and/or millimeters. Plank size is primarily an aesthetic specification — it determines the visual scale and proportion of the installed floor — but it also has practical implications for subfloor preparation requirements, carton coverage calculations, and layout planning. Reference-only: no product recommendations.

Quick answer

Common rigid core LVT plank widths range from 5 to 9+ inches; common lengths range from 36 to 72+ inches. Tile formats are typically 12×12 or 12×24 inches. Listed sizes are nominal — actual dimensions include manufacturing tolerances of ±0.2–0.5mm. Wider and longer planks require flatter subfloors. Plank size does not determine durability.

Common Plank and Tile Sizes

Plank format (wood-look)

Rigid core LVT plank products span a wide range of widths and lengths. Narrower strip planks (4–5 inches wide) replicate the look of traditional hardwood strip flooring. Standard planks (6–7 inches wide) represent the most common size range in the market. Wide planks (8–9 inches wide) create a more contemporary open appearance. Extra-wide planks (10–14 inches wide) are less common but increasingly available in premium collections. Length similarly varies: 36-inch planks are typical for entry-level products; 48-inch lengths are most common; 60-inch and 72-inch long planks are available in premium wide-plank products. The visual impression of a longer, wider plank is significant in open-plan spaces where the scale of the plank relative to the room size contributes to the design aesthetic.

Tile format (stone-look)

Tile-format rigid core LVT uses square or rectangular dimensions that replicate ceramic and porcelain tile proportions. The most common sizes are 12×12 inches (305×305mm), 12×24 inches (305×610mm), and 18×18 inches (457×457mm). Larger format tiles — 24×24 inches — are available in some premium lines. Tile-format rigid core LVT installs with the same click-lock system as plank products but the joint layout pattern is typically a grid or offset grid rather than a staggered plank pattern. The subfloor flatness requirements for large-format tiles are often tighter than for standard planks because the large span of each tile amplifies any subfloor undulation at joint edges.

Practical Implications of Plank Size

Plank length affects where end joints fall in the installation. Standard practice is to stagger end joints by at least 6–8 inches (and ideally one-third of plank length) between adjacent rows to prevent "H-joints" — four corners meeting at the same point — which create stress concentrations and a visually unnatural pattern. Longer planks make this stagger easier to maintain and produce a more random-looking layout. Plank width affects both the visual scale and the subfloor sensitivity: wider planks span more of the subfloor and are more likely to show lippage from flatness defects that narrower planks bridge over. When transitioning between areas, the plank size relative to the room width also affects how many cut pieces appear at walls and whether a full plank can be maintained as the starting row.

Spec Sheet Checklist

  • Confirm length and width in consistent units — some spec sheets list dimensions in inches, others in millimeters; convert to avoid calculation errors in takeoffs.
  • Check whether the listed size is nominal or actual — if tolerances matter (inlays, borders, precision grid patterns), request the dimensional tolerance specification.
  • Review subfloor flatness requirements for the specific plank size — wider or longer planks may require tighter flatness tolerances than standard.
  • Calculate carton coverage from the plank dimensions and pieces per carton to verify the spec sheet figure independently.
  • Consider the stagger requirement (minimum 6–8 inch end joint offset) relative to plank length when planning the layout starting configuration.

FAQ

Do bigger planks mean better quality? ⌄

No. Plank size is an aesthetic specification — it affects visual scale, not durability. A 9-inch plank and a 5-inch plank with the same wear layer, core type, and surface coating have essentially the same durability. Larger planks are popular because they create a more open visual impression, but they also require flatter subfloors and more careful installation to avoid lippage.

Are plank sizes nominal or exact? ⌄

Plank sizes are typically nominal — the design intent dimension, not the exact measured size. ASTM F3261 and EN ISO 24342 specify allowable dimensional tolerances, typically ±0.2–0.5mm. For most installations these tolerances are inconsequential, but for precision applications like detailed inlays or exact grid patterns, verify the actual dimensional tolerance from the product specification.

How does plank width affect subfloor requirements? ⌄

Wider and longer planks are more sensitive to subfloor flatness defects. A narrow plank may conform around modest undulations without visible lippage, while a 9-inch wide plank amplifies the same variation across its width. The standard 3/16 inch per 10 feet flatness requirement applies to all floating rigid core LVT, but manufacturers of wide-plank products sometimes specify tighter tolerances or additional prep requirements.

What is the difference between plank and tile format LVT? ⌄

Plank format LVT mimics wood board dimensions (4–9+ inches wide, 36–72+ inches long) for hardwood floor aesthetics. Tile format LVT mimics ceramic or stone tile dimensions (12×12, 12×24, 18×18 inches) for stone-look applications. Both formats use the same core types, locking systems, and wear layer grades. The choice is aesthetic; installation requirements for subfloor tolerance, expansion gaps, and transitions apply equally to both.
